### 安装

```bash
npm install @wisestCoder/translate -g
```

### 功能

- [x] 支持命令行使用
- [x] 中英互译（自动识别，无需指定源语言）
- [x] 翻译单词
- [x] 翻译短语
- [x] 翻译例句
- [x] 支持语音翻译（默认开启）

### 使用

```bash
fanyi hello

fanyi 你好
```
